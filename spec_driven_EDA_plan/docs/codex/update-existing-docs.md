# Existing documentation files that need updating

## Purpose

This document lists the older files from the first documentation pack that should be updated now that the workflow is TDD-first.

## Summary of change

The earlier order was implementation-led:

```text
eda_spec()
generate_synthetic_data()
check_schema()
profile_missing()
profile_summaries()
profile_plots()
run_eda()
render_eda_report()
project template
```

The revised order is TDD-first:

```text
external fixture plan
external fixture files
independently computed expected outputs
failing fixture-backed tests
implementation
repeat function by function
```

## TDD requirement

The MVP must be developed against external real-data fixtures.
